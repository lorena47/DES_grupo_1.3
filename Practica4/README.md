## Cómo ejecutar la web de Alejandría

1. Abre una terminal y navega a la carpeta del proyecto web:

   ```bash
   cd Practica4/alejandria-web
   ```

2. Instala las dependencias (solo la primera vez o si cambian):

   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:

   ```bash
   npm run dev
   ```

4. Abre tu navegador y accede a la URL que aparece en la terminal (normalmente http://localhost:5173).
